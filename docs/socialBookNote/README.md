# 社科类读书笔记

记录读书笔记的一些个人思考：

- 什么是读书？
首先读书的读，不仅仅局限于读。  
看书，发声读书，有声书听书，AI朗读听书， 视频网站拆书讲解视频等等方式获知书中内容都算“读”  
通过视频网站拆书等方式获知书中内容的方式属于二手知识，但是易于快速获取知识，可以通过这种方式先整体了解书中内容  
对书中内容感兴趣的话，一定要“读”一遍原著（看书、AI朗读听书等不经过他人进行内容加工的方式）  

另外当前短视频平台经常会有一些对书中部分概念的讲解视频  
这种视频虽然也能获取到一些知识，但对比整本书籍，这些知识点不够系统，也缺少一些论证，甚至一些会被灌输错误的观念  
因此不管任何形式的获取知识的方式，都需要辩证的看待这些观念，保持怀疑，小心论证。最后广泛的读书，结合实践验证，逐步形成属于自己的知识体系  

- 读不懂一本书怎么办

读书的过程中经常会遇到读不懂的书，其中有部分书看不懂是因为“未经历过”，“暂时用不到”等原因。  
一千个读者有一千个哈姆莱特，不同的人对同一本书的理解和解释是根据个人不同的经历与性格特质来的  
因此能读懂一本书，其中有一部分原因是因为你已经具备了书中的主题思想，读书只不过是更加系统的肯定和完善已有的观念。  
当然也有一部分书籍读起来让人眼前一亮，看到十分新颖的观点。认同或不认同，对作者观点进行辩证看待，小心求证，读到这种书籍对个人获益最大  

那么读不懂一本书的时候可以尝试看看图书拆解视频，看看别人的分析。若还是不懂，则直接放弃下一本即可。将来随着经历和认知的提升，再次重读时有一定的概率豁然开朗，读懂当初读不懂的书籍

### 自控力

#### 书籍简介

[《自控力》](https://book.douban.com/subject/10786473/)  
作者: 凯利·麦格尼格尔

本书讲述了什么是自控力，自控力如何发生作用，以及为何自控力如此重要  
书中提出了“我要做”“我不要”和“我想要”的力量是自控力的基础  
同时以锻炼肌肉力量作为比喻，提出了自控力也是可以锻炼的、但也存在极限，需要像分配体力一样使用的观点。

通过本书可以正确、深刻的理解自控力，但不要指望能短期掌握自控力。总体来说值得一读。

#### 个人观点

作者是斯坦福大学的心理学家，作者将自控力比作体力、肌肉表明自控力可以锻炼，但也可体力一样宝贵，类似“进食”恢复体力，及时释放压力有利于自控力的恢复。

个人认为仅将自控力用在对自己有意义的事情上，类似于体力耗尽但无法达到运动目标一样，对于使用自控力也无法达成目标的情况一样，不必抱有负罪感，坦然接受自己更有利于自控力的恢复，以便面临下次挑战。当然也仅有耗尽自控力的情况下才能“锻炼”自控力的强度。

对于作者提出的自控力通过“我想要”，“我不要”，“我要做”，“我应该”等途径的发挥作用的观点也十分认同，正是需要通过，“我不要”来达成拒绝诱惑，“我想要”让一切事情回归原本计划的轨道，“我要做”来推进计划，从而达成目标。

另外作者也提到自控力也有传染性，作为群体中的一员，自控力的强度也收到群体的影响。自控力也有副作用，过度使用容易形成自我否定的恶性循环，学会对内接受自我，对外控制行动才是维持和提升自控力的关键。

仅通过自控力达到自律从而走向成功是不可能的，想要培养某种习惯，学会某项技能更多的开始靠着热爱，那些能够长期坚持下来的习惯通常都是“热爱”在支撑。

类似书籍推荐[《心流：最优体验心理学》](#心流)，[《当下的力量》](#当下的力量)，[《练习的心态-如何培养耐心、专注和自律》](#练习的心态)，[《上瘾-让用户养成使用习惯的四大产品逻辑》](#上瘾)

### 丑陋的中国人

#### 书籍简介

[《丑陋的中国人》](https://book.douban.com/subject/1027247/)  
作者-柏杨

市面上存在[《丑陋的美国人》](https://book.douban.com/subject/1761842/)、[《丑陋的日本人》](https://book.douban.com/subject/3105926/)等书籍，批评对应民族文化的劣根性。本着每个民族文化都有其确认，柏杨先生写了丑陋的中国人一书。

书中强烈批判了，中华文化(包括大陆、台湾省、华侨华裔等在内)中存在的“脏，乱， 吵， 窝里斗， 不团结， 死不认错”等文化劣根性， 提出中华文化中的缺点“酱缸”易染的特点的观点。

批判中国不够民主、缠小脚等陋习

#### 个人观点

作者在书中提到写作初衷是揭露中华文化中的劣根性，想借此提醒国民改正缺点，推动文化进步。  
其中提出文化中存在“脏、乱、吵、窝里斗、不团结、死不认错、好面子、不民主、人情社会”等缺点的观点，切合实际，直击要害。  
但文中与美国、日本文化等进行对比时，提出了“崇洋媚外”的新的解释，即只“崇洋”学习外国文化的先进部分，而不“媚外”。  
但在具体分析时有拉踩之嫌。且文章仅提出问题，未给出解决方案，很难说在推动文化进步方面有多大作用。  
文末是当时各界大咖对柏杨观点的评判，有支持也有反对的，也是很精彩的一节。  

总的来说，接受批判，接受缺点，懂得反思改进，是进步的起点。  
书中批判的言辞激进，但切合实际，对文化进步有一定的积极意义。  
但没有提出建设性的改善意见，部分观点不够客观中立，在一定程度上限制了文章的影响力。  
在有独立思考能力的情况下，这本书还是值得一看。

## 心理学

- 为什么读心理学

每个人都为了适应其生活成长环境(主要是人文环境方面)， 都被动的学会了不同的性格特质（MBTI性格分类等等）。  
但现代社会快速变化的环境， 对个人性格形成了严峻的考验。  
当个人的性格特质对于新环境不适应时， 会造成诸如：偏执、焦虑、迷茫、拖延症、选择困难症、自我pua等自我内耗问题， 严重者甚至造成抑郁症。  
当代社会环境瞬息万变，因此这些问题或多或少、或轻或重的普遍存在于我们每个人身上。  

明确问题后，再来看看如何解决问题，在我看来如若遇到人生困境解决之道有两条：  
1. “改变”环境，比如逃离当前环境， 或具有足够能量时，改变当前环境  
2. 适应环境， 改变自身认知或性格特质，适应当前环境  

哪一条路都不好走，逃离当前环境，比如换一份工作，换一个城市代价都非常巨大， 而改变当前环境的能量更是大部分人所不具备的  
至于说适应环境，是一条相当艰辛的路，但相对容易一些。  

- 如何适应“环境”

个人看法是理论与实践相结合， 通过读书扩展认知，“拆掉思维里的墙”， 读一读心理学。运用心理学中的原理，解释过去的经历，观察周围人和事。  
找出适应当前环境的性格特质，理解并尝试按照这种特质思考和行事(最难的一步)，不断练习，从而达到适应环境的目的。

社会环境是不断变化的，因此上述过程也不会是一劳永逸的。我将这个过程称为自我成长，心智成熟之路。  
类比修行，想要心智更加成熟，我们都需要在人生路上修心，修行通天之路永无止境，而修心也是如此。  

- 如何通过心理学来适应环境

前面提到利用心理学的知识适应环境方式是，通过改变认知， “武装思想”的方式，从源头上修正想法与行为，属于“道”的层面。 这方面的书籍推荐有[《少有人走的路》](#少有人走的路)， [《亲密关系》](#亲密关系)，等等  
而利用实用的心理学技巧去指导具体情境的行为则是属于“术”的层面， 这方面包括的书籍有[《非暴力沟通》](#非暴力沟通)， [《自控力》](#自控力)， [《谈话的力量》](#谈话的力量)等等

### 依恋的形成

#### 书籍简介

[依恋的形成 : 母婴关系如何塑造我们一生的情感](https://book.douban.com/subject/27604891/)  
作者:凯伦（Karen, R.）

本书通过回溯早期经历、特别是母婴、父婴关系经历，试图弄清楚“我如何成了今天的我”，解释我们的依恋人格是如何形成的  

#### 个人观点

- 第一章： 孩子需要什么

提出孩子需要母爱的观点，哪怕是有严重缺陷的母爱， 也比没有母爱强

- 第二章： 走进鲍尔比：探究关系理论

讲述英国精神分析学家约翰·鲍尔比的核心研究方向， 童年经历与亲子关系对人格发展造成的影响，并强调了其重要性

- 第三章： 鲍尔比和克莱因：幻想与现实

讲述维也纳精神分析师克莱因的主要研究方向，早期心理过程理论，侧重于婴儿内心世界的幻想，即更关注于母婴关系的精神层面，并对鲍尔比产生了一定的影响

- 第四章： 精神病态者的形成：44个少年犯

本章提出宽容的养育氛围，允许孩子直接表达，引导发泄更加有利于孩子健康成长，形成自控力。  
并以44个盗窃少年犯的共性经历作为例证， 即他们都与母亲有过一段长时间的剥夺式分离经历(即妈妈离开了，但没有人接手，或者不断地更换抚育者)，最终形成了情感缺乏人格，极度冷漠的性格。  
虽然这些特质并不一定直接导致孩子犯罪，但对照组的心理健康的小孩一定没有这种经历。  
总之早期剥夺式的分离有害，令人痛苦并影响人格的形成  
这种观点让我联想到了中国的留守儿童的经历  

- 第五章： 战号吹响： 世界卫生组织报告

鲍尔比在研究中发现母爱剥夺儿童身上相似的症状：人际交往陷于表面，无法将心比心，难以接近，缺乏情绪回应，经常无谓的偷和骗，上课注意力不集中  
鲍尔比证明了母婴关系的极端重要性，它不是一项福利，而是一项绝对的必需品  
并提出了其研究核心母爱和母爱剥夺问题  

- 第6章：第1战场两岁儿童就医

本章以儿童就医为案例，引出了罗伯森在母婴分离问题上的研究成果及其理论观点，并讨论了这些研究对鲍尔比理论的影响，提出来母爱在剥夺过程中存在反抗、绝望和疏离这三个阶段的理论框架。

### 关系的重建

#### 书籍简介

[关系的重建](https://book.douban.com/subject/30320071/)  
作者: 阿米尔•莱文 / 蕾切尔•赫尔勒

本书讨论了依恋的类型，即安全型、焦虑型、回避型、焦虑回避型四种类型。  
简单解释了依恋的形成，并侧重于如何判别自己和他人的依恋类型。

#### 书籍简介

[被讨厌的勇气](https://book.douban.com/subject/36150914/)  
作者: (日) 岸见一郎，古贺史健

本书以年轻人与哲学家对话的形式总结了心理学精神分析学派三大巨头之一阿德勒的思想，与其他两位巨头的观点不同，其主要着眼于未来，对“人如何能够获得幸福”这个问题给出了简单且具体的回答。

#### 个人观点

这本书之前接触的都是弗洛伊德、荣格的作品， 阿德勒的观点则给人眼前一亮的感觉。  
弗洛伊德认为心理学“个性”主要源自童年经验和潜意识欲望的冲突。  
荣格则注重个体的自我探索与潜意识的潜意识的重要性，更加关注与个体的发展  
阿德勒更加强调个体的社会因素对心理发展的影响，以及如何超越个人困境来实现更高层次的成就和满足

简单的比喻来说，弗洛伊德关注的是归因，是“过去”，荣格关注是自我，是“当下”，阿德勒关注的是外部关系，是“未来”

### 钢铁是怎样炼成的

#### 书籍简介

[钢铁是怎样炼成的](https://book.douban.com/subject/1119908/)  
作者: [前苏联] 尼古拉·阿列克谢耶维奇·奥斯特洛夫斯基

小说以乌克兰内战和波苏战争为背景， 描写了主人公保尔·柯察金在为解放全人类的事业奋斗过程中遭遇的爱情，挫折等故事。  

#### 个人观点

小时候受舆论宣传听过了这本书，买了纸质版但因为各种原因没读。为了弥补遗憾，近来通过ai读书的方式听完了整本书。  
“读”完后，完全无法理解保尔因为阶级立场拒绝冬妮娅，对家庭亲情不管不顾等行为。或许正如书名，保尔逐步正在被练成钢铁， 而不是作为一个人。  
无疑，为伟大事业奋斗的保尔是伟大的， 值得歌颂的，但首先保尔必须作为一个“正常”的人， 而不是一个政治偏激、盲目服从，对己对人情感冰冷的“钢铁”， 才能称之为是一个伟大的人， 否则就是伟大的“钢铁”  
当然仅凭着“钢铁”的意志也是无法做成解放全人类这种伟大的事业的，中国及时变通，改革开放，走出特色社会主义道路取得了当前的成就就是最好的例证。  

抛开其他问题，这本书唯一值得学习的就只有保尔为了事业奋斗的那股精神，那股克服万般困难的钢铁般的意志。  
但这种钢铁般的意志只是一种成事的工具， 现实生活中拥有这种工具是个好事，但如何用好这个工具则是一个更大的课题  

总的来说，尽信书不如无书，每本书都需要去辩证的看待，取其精华去其糟粕。  
通过广泛阅读各种不同的书籍和观点，不走极端，逐步形成自己的独立思考能力和价值观
